# IOT OCCUPANCY MONITORING

## 📌 Project Description
This project focuses on developing an IoT-based occupancy monitoring system for energy-efficient smart buildings. It detects real-time human presence using sensors and automatically controls electrical appliances like lights and fans to reduce energy wastage.

---

## 🎯 Objective
- To detect occupancy using IoT sensors  
- To automate lighting and HVAC systems  
- To reduce energy consumption and operational cost  
- To enable real-time monitoring through IoT  

---

## 🚀 Features
- Real-time occupancy detection using PIR sensor  
- Automatic ON/OFF control of appliances  
- IoT-based remote monitoring  
- Energy-efficient smart automation  
- LCD display for live status  

---

## 🛠️ Technologies Used
- ESP32 / NodeMCU  
- PIR Sensor  
- Embedded C / Arduino  
- WiFi & MQTT Protocol  
- Cloud Server  
- Web Dashboard  

---

## ⚙️ System Working
1. Sensors detect human movement  
2. Data is processed by microcontroller  
3. Occupancy status is identified  
4. Appliances are controlled automatically  
5. Data is sent to cloud for monitoring  

---

## 📷 Results
- When occupancy is detected → Light & Fan ON  
- When no occupancy → Light & Fan OFF  
- Data displayed on IoT dashboard  
- Energy consumption reduced efficiently
