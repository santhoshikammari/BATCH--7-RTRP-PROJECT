# BATCH--7-RTRP-PROJECT
This project focuses on developing an IoT-based occupancy monitoring system to improve energy efficiency in smart buildings. In many buildings, electrical devices such as lights, fans, and air conditioning systems remain ON even when rooms are unoccupied, leading to unnecessary energy consumption and higher costs.

To address this issue, the proposed system uses sensors such as PIR sensors, cameras, or CO₂ sensors to detect human presence in real time. The sensor data is collected and processed using a microcontroller like Arduino or Raspberry Pi, and can be integrated with cloud platforms for monitoring and analysis.

Based on occupancy detection, the system automatically controls lighting and HVAC systems, ensuring that devices operate only when required. This reduces energy wastage, improves efficiency, and minimizes manual intervention.

Overall, the project demonstrates how IoT technology can be effectively used to create smart, automated, and energy-efficient building environments, contributing to cost savings and sustainable development.
